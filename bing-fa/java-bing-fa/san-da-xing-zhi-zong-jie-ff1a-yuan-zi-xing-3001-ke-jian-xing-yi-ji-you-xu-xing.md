## 2. 原子性

原子性是指\*\*一个操作是不可中断的，要么全部执行成功要么全部执行失败，有着“同生共死”的感觉\*\*。及时在多个线程一起执行的时候，一个操作一旦开始，就不会被其他线程所干扰。我们先来看看哪些是原子操作，哪些不是原子操作，有一个直观的印象：

```
> int a = 10;  //1
> 
> a++;  //2
> 
> int b=a; //3
> 
> a = a+1; //4
```
