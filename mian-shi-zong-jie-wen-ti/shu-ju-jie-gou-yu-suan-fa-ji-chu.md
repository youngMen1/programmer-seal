| 排序算法 | 平均时间复杂度 | 最差时间复杂度 | 空间复杂度 | 稳定性 |
| :--- | :--- | :--- | :--- | :--- |
| 冒泡排序 | O\(n²\) | O\(n²\) | O\(1\) | 稳定 |

## 2.用Java写一个冒泡排序算法

```
public void bubbleSort(int[] a) {
  if (a == null || a.length < 2) {
    return;
  }
  for (int i = a.length - 1; i > 0; i--) {
    for (int j = 0; j < i; j++) {
      if (a[j + 1] < a[j]) {
        int temp = a[j + 1];
        a[j + 1] =  a[j];
        a[j] = temp;
      }
    }
  }
}
```
